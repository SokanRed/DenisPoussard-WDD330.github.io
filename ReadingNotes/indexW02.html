▼ Question

Which value does x have after execution of the following code?

  • > 1 let x = ‘Laurel’;
  • > 2 let y = ‘Hardy’;
  • > 3 let z = y;
  • > 4 y = x;
  • > 5 x = z;
Answer : ‘Hardy’
Line 3 | y = ‘Hardy’
Line 4 |The value of y changes (y = ‘Laurel’)
Line 3 | z = y (z = ‘Laurel’)
Line 5 | x = z (x = ‘Laurel’)
But the answer is ‘Hardy’, Is the fact that line 5 goes back to line 3 to get the answer, don't consider what happens in line 4 ?

Answer

line 4 (y = x) does not change the value of z, even if the value of y changed
