▼ Chapter 3 | Arrays, Logic, and Loops

  • Arrays | mdn Array

    • An array is an ordered list of values. To create an array literal, simply write a pair of square brackets:

    const myArray = [];

    Initializing an Array

    const heroes = [];

    Adding Values to Arrays

    heroes[0] = 'Superman';

    Creating Array Literals

    const avengers = ['Captain America', 'Iron Man', 'Thor', 'Hulk'];
    >['Captain America', 'Iron Man', 'Thor', 'Hulk']

    Removing Values from Arrays

    delete avengers[3]; (will delete the item 'Hulk' from the array

    Destructuring Arrays

    Destructuring an array is the concept of taking values out of an array and presenting them as individual values.

    const [x,y] = [1,2]; (x = 1 & y = 2)

    Array Properties and Methods

    avengers.length;

    Pop, Push, Shift, and Unshift

    avengers.pop(); (will remove the last item from the array)
    avengers.shift(); (will remove the first item from the array
    vengers.push('Thor'); (will append a new item to the end of the array)
    avengers.unshift('Captain America'); (will append a new item at the beginnig of the array)

    Merging Arrays

    The concat() method can be used to merge an array with one or more arrays

    The join() Method

    The join() method can be used to turn the array into a string that comprises all the items in the array, separated by commas. You can choose a separator other than a comma by placing it inside the parentheses

    avengers.join(); or avengers.join(' & ');

    Slicing and Splicing

    The slice() method creates a subarray; effectively chopping out a slice of an original array, starting at one position and finishing at another. For example, if we wanted to find the 3rd and 4th item in our array we would use the following code. (Note-This operation is non-destructive ― no items are actually removed from the original array)

    The splice() method removes items from an array then inserts new items in their place. (Note-This is a destructive operation as it changes the value of the array

    avengers.splice(3, 1, 'Scarlet Witch');

    The first number in the parentheses tells us the index at which to start the splice. The second number tells us how many items to remove from the array.

    The splice() method can also be used to insert values into an array at a specific index without removing any items, by indicating that zero items are to be removed.

    avengers.splice(4,0,'Quicksilver');

    The splice() method is a particularly flexible method as it can be used to insert or remove values from an array. Be careful, though, as splice() is a destructive method which means it changes the array permanently.

    Reverse

    We can reverse the order of an array using the reverse() method. (Notes-This changes the order of the array permanently)

    Sort

    We can sort the order of an array using the sort() method. (Notes-This changes the order of the array permanently)

    We can use the indexOf() method to find the first occurrence of a value in an array. If the item is in the array, it will return the index of the first occurrence of that item.

    ES6 also introduced the includes() method. This returns a boolean value depending on whether the array contains a particular element or not.

    You can also add an extra parameter to indicate which index to start the search from :

▼ Chapter 4 | Functions | mdn Functions

  • Defining a Function

    • Function Declaration

    function hello(){
    console.log('Hello World!');
    }

    Function Expressions

    const goodbye = function bye(){
    console.log('Goodbye World!');
    };

    function() Constructor

    const hi = new Function('console.log("Hi World!");');
  • Invoking a Function

    • Invoking a function is to run the code inside the function’s body. To invoke a function, simply enter its name, followed by parentheses. This is how we’d invoke the hello function:

    hello();
    >> 'Hello world!'
  • Return Values

    • All functions return a value, which can be specified using the return statement, which comes after the return keyword. A function that doesn’t explicitly return anything (such as all the examples we have seen so far) will return undefined by default. The function in this example will return the string 'Howdy World!':

    function howdy(){
    return 'Howdy World!';
    }
  • Parameters and Arguments

    • Parameters and arguments are terms that are often used interchangeably to represent values provided for the function as an input. There is a subtle difference though: any parameters a function needs are set when the function is defined. When a function is invoked, it is provided with arguments. In this example, x is a parameter and 4 is an argument:

    function square(x){
    return x*x;
    }
    square(4);
    >> 16

    You can use as many parameters as you like when defining functions. Make sure that the number of arguments matches the number of parameters.

    Arrow Function

    const square = x => x*x;
      Arrow functions have a number of advantages over other ways of declaring functions:
    • They are much less verbose than normal function declarations.
    • Single parameters don't need putting into parentheses.
    • The body of the function doesn't need placing inside a block if it's only one line.
    • The return keyword isn't required if the return statement is the only statement in the body of the function.
    • They don't bind their own value of this to the function.
    const add = (x,y) => x + y;
    const hello = () => alert('Hello World!');

    Function Hoisting

    Hoisting is the JavaScript interpreter’s action of moving all variable and function declarations to the top of the current scope, regardless of where they are defined. Functions that are defined using a function declaration are automatically hoisted, meaning they can be invoked before they have been defined.

    Variable Hoisting

    Variable declarations that use the var keyword are automatically moved to the top of the current scope. It’s better practice to use const and let to declare any variables at the beginning of a block so hoisting is unnecessary.

  • Callbacks

    • A functions can also be given as a parameter to another function. A function that is passed as an argument to another is known as a callback.

  • Array Iterators

    • Arrays have a number of methods that utilize callbacks to make them more flexible.

    forEach()

    A for loop could be used to loop through each value in an array. An alternative is to use the forEach() method. This will loop through the array and invoke a callback function using each value as an argument. The callback function takes three parameters, the first represents the value in the array, the second represents the current index and the third represent the array that the callback is being called on.

    const colors = ['Red', 'Green', 'Blue']
    colors.forEach( (color,index) =>
    console.log(`Color at position ${index} is ${color}`) );
    >> "Color at position 0 is Red"
    >> "Color at position 1 is Green"
    >> "Color at position 2 is Blue"

    map()

    It is often used to process data returned from databases in array form, such as adding HTML tags to plain text. It returns a new array that replaces each value with the return value of the callback function.

    [1,2,3].map( square )
    >> [1, 4, 9]

    An anonymous function can also be used as a callback:

    [1,2,3].map( x => 2 * x);
    > > [2,4,6]

    The next example takes each item in the array and places them in uppercase inside paragraph HTML tags:

    ['red','green','blue'].map( color => `<p> ${color.toUpperCase()}</p>` );
    >> ['<p>RED</p>', '<p>GREEN</p>', '<p>BLUE</p>']

    Reduce()

    The reduce() method is another method that iterates over each value in the array, but this time it cumulatively combines each result to return just a single value.

    [1,2,3,4,5].reduce( (acc,val) => prev + val );
    >> 15

    Another example could be to calculate the average word length in a sentence: 1)The sentence can be converted into an array using the split() method, 2)Now we can use the reduce() function to calculate the total number of letters in the sentence, by starting the count at 0 and adding on the length of each word in each step, and 3) a simple division sum tells us the average word.

    const sentence = 'The quick brown fox jumped over the lazy dog'
    >> 'The quick brown fox jumped over the lazy dog'
    const words = sentence.split();
    >> ['The', 'quick', 'brown', 'fox', 'jumped', 'over', 'the', 'lazy', 'dog']
    const total = words.reduce( (acc,word) => acc + word.length,0 );
    >> 36
    length: const average = total/words.length;
    >> 4

    Filter()

    The filter() method returns a new array that only contains items from the original array that return true when passed to the callback.

    const numbers = [ 2, 7, 6, 5, 11, 23, 12 ]
    numbers.filter(x => x%2 === 0 ); // this returns true if the number is even
    >> [ 2, 6, 12 ]

    Chaining Iterators Together

    The various iterator functions can be used in combination to create some powerful transformations of data stored in arrays. For example, we can calculate the sum of square numbers using the map() method to square each number in the array and then chain the reduce() method on the end to add the results together:

    [1,2,3].map( x => x*x ).reduce((acc,x) => acc + x );
    >> 14

    Another more complex example could be used to take an array of orders, apply a sales tax to them using map() and then use reduce() to find the total:

    const sales = [ 100, 230, 55];
    totalAfterTaxSales = sales.map( (amount) => amount * 1.15 ).reduce( (acc,val) => acc + val );
    >> 442.75

    Improving the mean() Function

    function mean(array,callback) {
    if (callback) {
    array.map( callback );
    }
    const total = array.reduce((a, b) => a + b);
    return total/array.length;
    }
    mean([2,5,7,11,4]); // this should just calculate the mean
    >> 5.8
    mean([2,5,7,11,4],x => 2*x); // An anonymous arrow function is used to double all the numbers before calculating the mean
    >> 11.6
